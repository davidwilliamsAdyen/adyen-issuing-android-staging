fun create(applicationCertificate: X509Certificate, paymentInstrumentIds: Set<String>, activityProvider: () -> Activity, sessionTokenProvider: suspend (Set<String>) -> String, environment: ProvisioningEnvironment = ProvisioningEnvironment.LIVE): ProvisioningSession

Provides an instance of ProvisioningSession using the given X509Certificate object as the application certificate.

Return

A ProvisioningSession instance.

Parameters

applicationCertificate

A certificate provided to the client by Adyen and unique to each client application.

paymentInstrumentIds

The payment instrument ids of the cards that may be provisioned during the session.

activityProvider

A function which returns an Activity. The Activity returned should be the Activity which hosts the card provisioning functionality within the client application.

sessionTokenProvider

a function implemented by the client which returns a session token via the client backend and originating from the Adyen backend session provider.

environment

optional parameter allowing selection of a ProvisioningEnvironment. Default is ProvisioningEnvironment.LIVE.
